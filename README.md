# Full-Stack Web Development Portfolio

A responsive, professional portfolio website showcasing modern front-end layout styling combined with verified back-end server logic.

## 🚀 Features Build
- **Responsive Navigation & Layout:** Structured with Bootstrap 5 elements for clean views across mobile and desktop devices.
- **Dynamic Contact Form:** Implemented inside a Bootstrap modal overlay with interactive invalid validation feedback messages.
- **Client-Side Validation:** Scripted with Vanilla JavaScript to intercept form submissions and verify field constraints locally.
- **Server-Side Security PHP Processing:** Features robust XSS sanitization (`htmlspecialchars`), post-request constraint enforcement, and field completion empty checks.

## 🛠️ Tech Stack Applied
- **Front-End:** HTML5, CSS3 (Custom Root Theme Variables), Bootstrap 5 CDN
- **Interactivity:** JavaScript (ES6 Modules)
- **Back-End:** PHP 8.x Form Handler
- **Version Control:** Git & GitHub Branches Workflow
